# grachio-ui

http://grachio.s3-website-us-east-1.amazonaws.com/

## Project setup
```
yarn install
```

### Compiles and hot-reloads for development
```
yarn serve
```

### Compiles and minifies for production
```
yarn build
```

### Lints and fixes files
```
yarn lint
```

### Run your unit tests
```
yarn test:unit
```

### Deploy to S3
```yarn deploy```
